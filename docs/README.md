# Bank Tool

> 一款银行自动转账及查账辅助操作（自动化）软件

## 简介


Bank Tool 支持API 对接,通过API 接口下发 转账、查询命令，
PC辅助工具自动完成，转账、查询操作并且通知服务器结果
