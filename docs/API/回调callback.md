# 4.回调callback



## URL

下单接口  call_back参数   由第三方平台 自定义



## 请求方式   GET
 
 
### 请求参数说明

| 参数        | 含义           | 备注  |
| :------------- |:-------------:| :-----|
| userid      | 用户id   个人中心->userid    |   |
| order_no     | 订单号      |    |
| trade_no    | 第三方平台订单号      |   |
| payerid     | 付款人id      |    |
| status     | 订单状态      |  1：待支付； 2：支付完成； 3：已回调,4：已结算,5：失败,6：订单异常 |
| sign    | 签名      |   参考文档签名 介绍  |


		
```

status 订单状态  需要处理 2：支付完成；5：失败

```		
		

## 返回参数
```$xslt
{
    "msg": "请求成功",
    "code": 200,
    "info": {}
}
```
code  为 200
认为回调成功，其他都认为失败
